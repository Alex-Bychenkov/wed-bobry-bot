# Настройка прав администратора для бота

## Почему боту нужны права администратора?

Бот **ОБЯЗАТЕЛЬНО** должен иметь права администратора в группе для корректной работы функции автоматического удаления сообщений.

### Что может удалять бот БЕЗ прав администратора:
- ✅ Только свои собственные сообщения

### Что может удалять бот С правами администратора:
- ✅ Свои сообщения
- ✅ Сообщения пользователей (команды /start, /status, /reset, /close)
- ✅ Сообщения с фамилиями пользователей
- ✅ Любые другие сообщения в группе

## Как добавить бота в администраторы группы

### Шаг 1: Откройте настройки группы
1. Откройте чат группы в Telegram
2. Нажмите на название группы вверху
3. Выберите "Редактировать"

### Шаг 2: Добавьте бота в администраторы
1. Нажмите "Администраторы"
2. Нажмите "Добавить администратора"
3. Найдите и выберите бота `@Bobry_Mytishchi_Bot`

### Шаг 3: Настройте права бота
**Минимально необходимые права:**
- ✅ **Удаление сообщений** - ОБЯЗАТЕЛЬНО для работы автоудаления
- ✅ **Закрепление сообщений** - для функции закрепления списка участников

**Опциональные права (можно отключить):**
- ❌ Изменение информации о группе
- ❌ Приглашение пользователей
- ❌ Блокировка пользователей
- ❌ Добавление администраторов
- ❌ Управление видеочатами
- ❌ Остальные права

### Шаг 4: Сохраните изменения
Нажмите "Готово" или "Сохранить"

## Проверка работы

После добавления бота в администраторы:

1. Отправьте команду `/start@Bobry_Mytishchi_Bot`
2. Команда должна удалиться через 3 секунды
3. Появится сообщение с кнопками
4. При повторном вызове `/start` старое сообщение с кнопками удалится

## Текущее поведение автоудаления

| Сообщение | Время удаления | Требует прав |
|-----------|----------------|--------------|
| Команда `/start` | 3 сек | ✅ Да |
| Команда `/status` | 3 сек | ✅ Да |
| Команда `/reset` | 3 сек | ✅ Да |
| Команда `/close` | 3 сек | ✅ Да |
| Сообщение с кнопками | При следующем `/start` | Нет (свое сообщение) |
| "Сессия сброшена" | 3 сек | Нет (свое сообщение) |
| "Сессия закрыта" | 3 сек | Нет (свое сообщение) |
| "Фамилия сохранена..." | 3 сек | Нет (свое сообщение) |
| "Участник добавлен..." | 3 сек | Нет (свое сообщение) |
| Сообщение пользователя с фамилией | 3 сек | ✅ Да |
| "Пожалуйста, отправь фамилию" | 15 сек | Нет (свое сообщение) |
| "Введите фамилию участника..." | 15 сек | Нет (свое сообщение) |

## Что произойдет, если НЕ дать права администратора?

Без прав администратора бот будет продолжать работать, но:
- ❌ Команды пользователей `/start`, `/status`, `/reset`, `/close` НЕ будут удаляться
- ❌ Сообщения пользователей с фамилиями НЕ будут удаляться
- ✅ Свои сообщения бота будут удаляться корректно
- ✅ Основной функционал бота (сбор ответов) будет работать

**Вывод:** Для чистоты чата и корректной работы автоудаления - права администратора ОБЯЗАТЕЛЬНЫ.
