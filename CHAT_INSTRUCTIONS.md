# Инструкция для закрепления в чате

Скопируйте этот текст и закрепите в чате группы для удобства участников.

---

## 📝 Текст для закрепления:

```
🏀 КАК ПОЛЬЗОВАТЬСЯ БОТОМ БОБРЫ

👥 ДЛЯ УЧАСТНИКОВ:

1️⃣ Отметить участие:
   /start → Нажми кнопку с твоим статусом
   🟢 Буду
   🟡 Не определился
   🔴 Не смогу

2️⃣ Посмотреть список:
   /status

3️⃣ Изменить решение:
   /start → Нажми другую кнопку

💡 Все служебные сообщения автоматически удаляются для чистоты чата

---

👨‍💼 ДЛЯ АДМИНИСТРАТОРОВ:

/reset — Сбросить сессию (новая встреча)
/close — Закрыть сессию (завершить сбор)

➕ Добавить гостя:
   /start → Кнопка "➕ Добавить участника" → Ввести фамилию

---

📅 АВТОМАТИЧЕСКИЕ НАПОМИНАНИЯ:

• Среда и суббота в 11:00 — Напоминание о встрече
• Среда в 23:30 — Автоматическое закрытие сессии

---

Вопросы? Смотри документацию: https://github.com/your-repo
```

---

## Альтернативный краткий вариант:

```
🏀 БОТ БОБРЫ - ИНСТРУКЦИЯ

Отметить участие: /start
Посмотреть список: /status

🟢 Буду | 🟡 Не определился | 🔴 Не смогу

Админы: /reset (новая сессия) | /close (закрыть)

📅 Напоминания: Ср и Сб в 11:00
```
